\hypertarget{index_autotoc_md1}{}\doxysection{Présentation du TP}\label{index_autotoc_md1}
Ce projet a pour but de créer une bibliothèque permettant de simplifier la programmation de jeux utilisant des tableaux de taille variable. Cette bibliothèque sera utilisée pour implémenter le jeu des allumettes. Le projet est réalisé dans le cadre du cours de programmation en C de l\textquotesingle{}Université de Lorraine.\hypertarget{index_autotoc_md2}{}\doxysection{Jeux des allumettes}\label{index_autotoc_md2}
\hypertarget{index_autotoc_md3}{}\doxysubsection{Présentation du jeux}\label{index_autotoc_md3}
Le jeu des allumettes est un jeu de réflexion se jouant à deux joueurs. Le jeu se joue avec n allumettes au départ. Les joueurs jouent à tour de rôle. À son tour, un joueur doit enlever entre une et trois allumettes. Le joueur qui prend la dernière allumette a perdu.\hypertarget{index_autotoc_md4}{}\doxysection{Exemple de jeu qui peut utiliser cette bibliothèque}\label{index_autotoc_md4}
\hypertarget{index_autotoc_md5}{}\doxysubsection{La bataille navale}\label{index_autotoc_md5}
La bataille navale est un jeu pour deux joueurs. Chaque joueur place ses navires sur une grille de dimensions généralement 10x10, sans que l\textquotesingle{}adversaire ne les voit. Les joueurs tirent tour à tour des missiles sur les cases de la grille ennemie en tentant de toucher les navires adverses. Si un joueur touche une case occupée par un navire ennemi, il peut continuer à tirer jusqu\textquotesingle{}à ce qu\textquotesingle{}il manque sa cible. Le premier joueur à couler tous les navires ennemis remporte la partie.
